  2. My second project is coded in Java and it is Exam Module, designed for computerized tests for different subjects on the scheduled day. The student needs to enter the student number and password and can take the test.

    Neither the student can attempt test before the scheduled exam day.
  3. The next project (Heart Disease prediction ) is also coded in Java and is based on Data Mining concept(Naive Bayes). The project is designed from the begineer's prospective.
    The user needs to enter data to check whether he/she has possibility to have heart disease in future or not.
    The prediction is done on basis of already stored data of the heart patients.
