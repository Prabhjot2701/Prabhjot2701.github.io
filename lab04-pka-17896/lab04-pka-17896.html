What are responsive pages!!!

Responsive design means creating web pages that adapt to different devices and screen sizes. It uses the same content but presents it differently depending on whether you're using a mobile phone, iPad, laptop or desktop computer.

Desktops and Laptops

The desktop layout used these days is 960px wide(minimum) and it fit desktop computer screens as well as larger laptops.
Moreover, it will work for tablets with landscape mode.

Tablets

For tablets, the minimum width can be chosen as 768px. These days different devices like iPad-mini or Samsung Galaxy Note etc. have similar screen, so this width is appropriate for such devices.

Mobile Phones

With time, new mobile devices come into market. So the minimum width that can be used is 320 px for portrait mode.
