CSIS 1280-Multimedia Web Development

Why to choose Multimedia Web Development???

This course covers the fundamentals of Web site development and design using HTML, CSS and JavaScript and will cover following content:

  • Understanding fundamental concepts of client-server models, and role of client-side framework in design and creation of websites.
  • Working with tables and columns and Designing web forms.
  • Enhancing a website with multimedia, animations and transitions.
  • Creating responsive websites using HTML and CSS.

What YOU will learn?

  • how to create structured websites using HTML
  • how to use the most up to date CSS styles to create responsive, visually-interesting pages and captivating graphical designs
  • how to implement client-side script using basic concepts in JavaScript to access DOM elements and to validate web forms
